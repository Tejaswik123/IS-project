# KrishiGuard AI

KrishiGuard AI is a mobile-first agriculture and environment intelligence demo for the Indian Servers AI Model Development Contest agriculture/environment track.

## Features

- AI Crop Doctor workflow with image upload, crop/symptom inputs, confidence, reasoning, limitations, and expert warning.
- Advisory chat for crop disease, irrigation, soil health, waste management, and pollution guidance.
- Soil health and smart irrigation calculator using moisture, rain forecast, pH, and soil type.
- Environment assistant for waste classification and sustainability scoring.
- English, Hindi, and Telugu hero-language toggle.
- Contest-ready model/API explanation section for demo videos.

## Run

Open `index.html` in a browser.

## Future AI Integration

The current app is a polished frontend prototype with deterministic demo logic. To make it production-grade, connect:

- Vision: YOLO, CNN, Gemini Vision, Qwen-VL, or OpenAI vision for crop disease/pest detection.
- Advisory: RAG over agriculture extension manuals, crop packages, local weather rules, and regional-language prompts.
- Weather: live weather/environment APIs for rainfall, humidity, heatwave, and disease-risk prediction.
- Safety: agronomist-reviewed pesticide rules, local dosage references, confidence thresholds, and escalation flows.
